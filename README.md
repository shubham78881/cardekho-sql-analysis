# CarDekho Sales Data Analysis using SQL

## Overview
This project analyzes the CarDekho dataset using MySQL and covers data exploration, pricing trends, sales analysis, mileage analysis, and advanced SQL concepts.

## SQL Concepts Used
- GROUP BY
- ORDER BY
- Aggregate Functions
- CASE WHEN
- Window Functions
- CTEs
- Correlated Subqueries

## Project Highlights
- 30+ SQL Queries
- Price Analysis
- Demand Analysis
- Mileage & Engine Insights
- Brand-wise Performance Analysis
